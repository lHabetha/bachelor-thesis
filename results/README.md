# Results

Frozen thesis-facing summaries, generated figures/tables, checkpoints, and
outputs live here, organized by chapter. Runnable code belongs in `../code/`;
released datasets belong in `../datasets/`.

Checkpoints are stored here as frozen thesis-facing reproduction inputs because
they are tied to reported result tables, even though later scripts may load them.

- `chapter3_clevis_setup/`: setup and labeling summaries for the modified clevis.
- `chapter4_label_efficiency/`: label-efficiency outputs and model-comparison
  summaries reported in Chapter 4.
- `chapter5_optimization/`: tables, summaries, checkpoints, and exported outputs
  for Chapter 5 optimizer results.
- `chapter6_overlap/`: overlap-regression and overlap-repair artifacts reported
  in Chapter 6.
- `chapter7_aeroforge/`: AeroForge overlap and performance-preserving repair
  artifacts reported in Chapter 7.
