# Datasets

Released datasets and labeled pools live here, organized by thesis chapter.

Each dataset folder contains a README that documents:

- the thesis chapter/table/figure anchors it supports;
- file format and schema;
- generation or filtering rules;
- license or redistribution notes.

- `chapter4_clevis_pools/`: candidate pools and seed sets for the Chapter 4
  label-efficiency experiments.
- `chapter5_blocked200/`: the blocked-start benchmark used by the Chapter 5
  optimizers.
- `chapter6_overlap_clevis/`: overlap pools and strict-repair benchmarks for
  Chapter 6.
- `chapter7_aeroforge_adv/`: the AeroForge ADV pool, labels, and repair
  benchmark for Chapter 7.
