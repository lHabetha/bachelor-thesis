# Code

The thesis Python code lives here, organized by chapter. Imports, data paths,
and output paths are set up to work inside this repository.

- `chapter3_clevis_setup/`: clevis generation, validity constraints, proposal
  streams, and analytic assemblability labels.
- `chapter4_learning/`: MLP, MPNN/graph, tree, and active-learning training and
  evaluation code.
- `chapter5_optimization/`: assemblability repair optimizers and table/export
  scripts.
- `chapter6_overlap/`: overlap labeling, overlap model training, strict repair,
  and alternating overlap/assemblability pipeline code.
- `chapter7_aeroforge/`: minimal AeroForge ADV, CAD/overlap, quick-simulation,
  surrogate, and repair code needed by the thesis.
